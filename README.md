# SSA_Alias_game

Шаблон серверной части для телегорам бота (в данном случае игры)

Пишем t.me/botfather — это служебный бот Telegram для создания других ботов. 
Первая команда, которая нам нужна, — /newbot, следуем инструкциям и получаем access-токен.

Выбираем название бота (можно с пробелами)

Выбираем username бота (без пробелов и обязательно Bot)

Сохраняем User tocken ACCESS key вида 123456789:SDFGH402R2F2C23-C23RC2

/setinline  - включаем возможность делиться ботом

/newgame - создаем игру
попросит загрузить картинку 640x360px
gif можно пропустить


Переименовываем .env.example в .env
Заполняем файл полученными данными
TELEGRAM_API_TOKEN="123456789:SDFGH402R2F2C23-C23RC2"
GAME_URL="https://myMegaGame.ru"
GAME_NAME="MegaGame"
